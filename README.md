   ###安装
   ~~~bash
   #最新版本确认
   $ npm show vue-cli
   $ [sudo] npn install -g vue-cli@2.9.6
   
   #安装版本确认
   $ vue -V
   
   #命令帮助
   $ vue -h
   ~~~~
